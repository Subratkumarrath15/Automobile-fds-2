# Automobile Dataset Analysis

## Project Overview

This project performs exploratory data analysis on the Automobile Dataset using Python.

## Features

- Read and inspect dataset
- Identify attribute types
- Calculate dissimilarity measures
- Generate data visualizations
- Save graphs automatically

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Project Structure

```
Automobile-fds-2
│
├── Automobile_data.csv
├── read.py
├── identify_att.py
├── dissimilarity.py
├── visualization.py
├── requirements.txt
├── README.md
└── graphs/
```

## Graphs Generated

- Fuel Type Distribution
- Body Style Distribution
- Price Histogram
- Engine Size vs Price
- Average Price by Number of Cylinders
- Horsepower Box Plot

## How to Run

```bash
pip install -r requirements.txt
python read.py
python identify_att.py
python dissimilarity.py
python visualization.py
```
